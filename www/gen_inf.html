General Information

T61 is a development specific for the NA61 experiment of NA49 ROOT mini-DST classes written by Gunther Roland.

T61 classes are divided into two libraries.

One (T61DST) is containing all classes that are used for storing the DSTs in the mini-DST trees and consists basically of the ROOT61-classes with a number of extensions.
The extensions to ROOT61 include on one side additional data-members for each track (see T49ParticleRoot), which now also contain, e.g., the first and last coordinate of a track in each TPC and the PID, and on the other side classes for the storage of secondary vertices (T49VertexRoot) and Monte Carlo Information (T49EventMCRoot, T49ParticleMCRoot and T49VertexMCRoot).

The second library (T61ANA) provides all kinds of tools (e.g. event and track cuts, event mixing, dEdx-analysis, etc.) to be used in the analysis of the ROOT mini-DST.

These two libraries can either be directly loaded via

gSystem->Load("libT61DST.so") and
gSystem->Load("libT61ANA.so")

into the standard ROOT, or one might also use the root61-executable that has all these classes already included.
The root61 additionally contains the TRootDS-class, originally written by Gunther Roland, which allows a direct access to NA61-data in DSPACK format.
This way root61 has the same functionality as the former rootds.
